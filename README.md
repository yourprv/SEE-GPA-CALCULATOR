# SEE GPA Calculator

A lightweight, client-side web application designed for Nepali students to quickly calculate their Secondary Education Examination (SEE) Grade Point Average. Built with pure HTML and CSS for maximum speed and accessibility.

## 🚀 Features
- **Instant Calculation:** Real-time GPA generation based on subject grades.
- **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.
- **Privacy First:** All calculations are performed locally in the browser; no data is collected or stored.
- **Simple UI:** Minimalist interface for ease of use during the result season.

## 🛠️ Usage
1. Open `index.html` in any modern web browser.
2. Select or enter your grades for each subject.
3. The GPA will be calculated automatically.
